WeightedOnlineStats.jl

A version onf OnlineStats.jl allowing observations with custom weights.

Basics

Creating

julia> using WeightedOnlineStats
julia> m = WeightedMean()WeightedMean: ∑wᵢ=0.0 | value=0.0
julia> x = randn(100);
julia> w = randn(100);

Updating

julia> fit!(m, x, w)WeightedMean: ∑wᵢ=3.57047 | value=-1.59652

Merging

julia> m2 = WeightedMean()WeightedMean: ∑wᵢ=0.0 | value=0.0
julia> x2 = rand(100);
julia> w2 = rand(100);
julia> fit!(m2, x2, w2)WeightedMean: ∑wᵢ=46.7357 | value=0.474723
julia> merge!(m, m2)WeightedMean: ∑wᵢ=50.3061 | value=0.327717
